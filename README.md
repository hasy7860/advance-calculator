# Advance Calculator

A simple command-line calculator with three modes:
basic operations, percentage, and interest.

**Features**
- Basic math: add, subtract, multiply, divide
- Percentage calculator
- Simple interest calculator
- Menu-driven CLI

**Project Structure**
- `main.py` - entry point and menu loop
- `calculator/basic_calc.py` - basic math functions
- `calculator/percentage_calc.py` - percentage calculator
- `calculator/interest_calc.py` - interest calculator

**How To Run**
```bash
python main.py
```

**Usage**
1. Choose a mode:
   - `basic` for basic math
   - `p` for percentage
   - `i` for interest
   - `exit` to quit
2. Follow the prompts.

**Example**
```text
Enter choice: basic
---Type one of it (+, -, *, /) to calculate---
Enter here: +

Enter number 1: 10
Enter number 2: 5

15
```
